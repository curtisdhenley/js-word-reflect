# Word Reflect: A Coding Challenge

Word Reflect is a straightforward web application that takes a user-provided word, reverses it, and displays the result on the page. This project showcases fundamental JavaScript skills, including loops, DOM manipulation, and boolean logic.

## Features

- **Word Reversal:** Reverses the user-provided word and displays the result on the page.
- **Tech Stack:** Built with JavaScript, Bootstrap, HTML, and CSS.

## Tech Stack

- **JavaScript:** The core language for dynamic and interactive functionality.
- **Bootstrap:** Provides a clean and responsive layout.
- **HTML and CSS:** Essential components for structuring and styling the web app.

## Usage

Visit the live application: [Word Reflect Live](https://ch-word-reflect.netlify.app)

Experience the instant reversal of any word you input.
